# 初中数学

## 说明
自己业余时间用来给自家娃讲解的例题以及作业。
非专业教师，时间和能力都有限，无证上岗，自娱自乐。


## 已包含内容
### 六年级下
* U1 整式恒等变换1
* U2 因式分解：双十字相乘和主元法
* U3 因式分解：换元法 
* U4 因式分解：试根法，因式定理，待定系数法
* U5 轮换式和对称式
* U6 整式恒等变换2


## 编译环境
* TeX Live 2019
* BHCexam v1.6

XeLaTeX 编译
